# Github Pages

- Date: 01/27/2023

## Process

- create a repo named "username.github.io"
- run ```jekyll new mysite```
  - move the files generated by the above command to the ```username.github.io``` folder
  - remove ```mysite```

### Using Homebrew to install Ruby

- **This ended up not working on my current MacBook**
- Homebrew is a free and open-source software package management system that simplifies the installation of software on Apple's operating system, macOS, as well as Linux
- requirement
  - Ruby
  - Gem: a package manager for the Ruby programming language
- After upgrading to MacOS Ventura (13.0.1)
  - ```gem``` uses ```universal-darwin22``` to compile files
    - we need to create a link to the existing ```universal-darwin21```
      - to perform this action, we need to enable Terminal to update or delete other applications
        - Settings > Privacy & Security > App Management > Terminal
      - ```cd /Applications/Xcode.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX.sdk/System/Library/Frameworks/Ruby.framework/Versions/2.6/usr/include/ruby-2.6.0/```
      - ```sudo ln -sf universal-darwin21 universal-darwin22```
- Another requirement (this probably differs based on different devices)
  - **[export GEM_HOME](https://github.com/rbenv/rbenv/issues/1267)**
    - ```export GEM_HOME="$HOME/.gem"```
